# Vengeance Systems

> "If it can be optimized, it's not done."

We build lightweight, high-performance software with a strict focus on zero-dependency architecture and long-term stability. No bloated frameworks, no unnecessary abstractions.

---

### Active Projects

#### 🔹 Sentinel Flow
A secure, framework-agnostic Single Page Application (SPA) for enterprise process management. 
* Features a custom Gantt diagram editor, modular architecture, and session-isolated RBAC.
* Engineered entirely in vanilla PHP and native JavaScript with no transpilers, bundlers, or third-party wrappers.
* **Current State:** `v2.4.0` (Production stable)

#### 🔹 Synapse
A metacognitive knowledge graph and Model Context Protocol (MCP) data vault.
* Designed to bridge local technical documentation with LLM orchestration for autonomous development.
* Enforces strict frontmatter tagging and automated memory-retrieval protocols to ensure no knowledge gap survives a session.
* **Current State:** `v0.11.0` (Active development)
